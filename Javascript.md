### Tipos de Variáveis
**Pode ser acessada de qualquer parte do código**

``
var x = 5;
``

**Funciona apenas dentro de um escopo**

``
let x = 5;
``

**Let que não pode ser alterado**

``
const x = 5;
``

### Operações de Atribuição 

``
+= é o mesmo que x = x + y
``

``
-= é o mesmo que x = x - y
``

``
*= é o mesmo que x = x * y
``

``
/= é o mesmo que x = x / y
``

``
%= é o mesmo que x = x % y
``

``
**= é o mesmo que x = x ** y
``

``

``

``

``

``

``
