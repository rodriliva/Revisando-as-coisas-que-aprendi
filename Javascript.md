### Tipos de Variáveis
**Pode ser acessada de qualquer parte do código**
``
var x = 5;
``
**Funciona apenas dentro de um escopo**
``
let x = 5
``
**Let que não pode ser alterado**
``
const x = 5;
``

### Operações de Atribuição 

``
+= é o mesmo que x = x + y
-= é o mesmo que x = x - y
*= é o mesmo que x = x * y
/= é o mesmo que x = x / y
%= é o mesmo que x = x % y
**= é o mesmo que x = x ** y
``
## Operações de Atribuição 

``
==	Valor igual
===	Valor e Tipo igual
!=	Não Igual
!==	Não é igual de valor e tipo
>	Maior que
<	Menor que
>=	Maior ou igual 
<=	Menor ou igual
?	ternary operator
``

## Operadores Lógicos
``
&&	Lógico E
||	Lógico Ou
!	Lógico Não é igual
``
## Operações de Tipo

typeof	Returns the type of a variable
